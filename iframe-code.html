Iframe-Code für Terrassendach Konfigurator

Kopieren Sie den folgenden Code und fügen Sie ihn in Ihre Onlineshop-Startseite ein:

<iframe src="https://IHR-DOMAIN.de/embed.html" width="100%" height="800px" frameborder="0" style="border: none; min-height: 600px;" allow="fullscreen" title="Terrassendach Konfigurator" ></iframe>
Hinweis: Ersetzen Sie IHR-DOMAIN.de mit Ihrer tatsächlichen Domain, auf der der Konfigurator gehostet wird (z.B. terrassendach-konfigurator.vercel.app).

Responsive Variante (empfohlen)

Für eine bessere mobile Ansicht verwenden Sie diese Variante:

<div style="position: relative; width: 100%; padding-bottom: 75%; height: 0; overflow: hidden;"> <iframe src="https://IHR-DOMAIN.de/embed.html" style="position: absolute; top: 0; left: 0; width: 100%; height: 100%; border: none;" frameborder="0" allow="fullscreen" title="Terrassendach Konfigurator" ></iframe> </div>

Mit fester Höhe (einfachste Variante)

Wenn Sie eine feste Höhe bevorzugen:

<iframe src="https://IHR-DOMAIN.de/embed.html" width="100%" height="800px" frameborder="0" scrolling="no" style="border: none; display: block;" allow="fullscreen" title="Terrassendach Konfigurator" ></iframe>

Anpassungen

  • Höhe ändern: Passen Sie den Wert height="800px" an Ihre Bedürfnisse an
  • Breite: width="100%" passt sich automatisch an den Container an
  • CSS-Styling: Sie können zusätzliche CSS-Klassen oder Styles hinzufügen

Shop-System spezifische Hinweise

  • Shopify: Fügen Sie den Code in einen "Custom HTML"-Block ein
  • WooCommerce: Verwenden Sie einen "Custom HTML"-Block oder Shortcode
  • Shopware: Verwenden Sie einen HTML-Content-Block
  • Magento: Fügen Sie den Code in einen CMS-Block mit HTML ein
